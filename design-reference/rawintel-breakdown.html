Game Shape

This is a pace control game. Denver wants to slow things down — run half-court sets, let Jokić operate in space, grind possessions. Minnesota wants the opposite — push the pace, create chaos in transition, make this a track meet.

The game shape is defined by whoever wins that battle. If Denver controls tempo, this is a low-scoring, tight execution game. If Minnesota gets out in transition, the total climbs and variance spikes.
